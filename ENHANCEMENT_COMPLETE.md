## 🚀 如何使用

### 启动应用
```bash
npm run dev
```

### 测试步骤

1. **打开图纸编辑器**
   - 访问应用
   - 点击"图纸编辑"模式

2. **上传测试图片**
   - 准备一张拼豆图纸图片
   - 拖拽或点击上传

3. **调整识别参数**
   - 在左侧面板找到"🔍 增强识别"部分
   - 根据图片特点选择合适的算法

4. **查看识别结果**
   - 系统会显示检测质量（优秀/良好/一般）
   - 检查识别的豆子数量和色号是否准确

### 推荐配置

#### 场景1：规则排列的清晰图纸
```
边界检测：自适应 或 网格检测
颜色采样：中位数 或 平均值
颜色匹配：加权RGB
排除边缘：关闭
```

#### 场景2：复杂布局或有阴影的图纸
```
边界检测：轮廓检测
颜色采样：中位数 或 主导色
颜色匹配：CIELAB
排除边缘：开启
```

#### 场景3：低质量或模糊图片
```
边界检测：自适应
颜色采样：中位数
颜色匹配：加权RGB
排除边缘：开启
```

## 📊 技术亮点

### 边界检测改进
- 使用**众数**而非平均值估算网格间距（更抗噪声）
- **多行/列采样**提高间距估算准确性（5行采样）
- **精确边界定位**：向四个方向扩展直到颜色差异超过阈值
- **洪水填充**算法用于轮廓检测（支持不规则形状）

### 颜色识别改进
- **排除边缘像素**：默认排除20%边缘区域
- **中位数采样**：对每个RGB通道分别计算中位数
- **主导色算法**：使用16级量化减少噪声
- **CIELAB色彩空间**：最接近人眼感知的颜色距离计算

### 质量评估
- 返回检测置信度（0-1）
- 根据置信度自动选择算法（自适应模式）
- 向用户反馈检测质量（优秀>70%，良好>50%）
