### 问题2：左侧操作栏内容过多，无法滚动 ✅ 已修复

**修改内容**：
- 添加了 `ScrollArea` 组件包裹中间内容
- 头部（标题和返回按钮）固定不滚动
- 底部（导出按钮）固定不滚动
- 中间所有配置选项可以滚动访问

**修改位置**：`src/pages/BlueprintEditor.tsx`
- 第1308行：添加 `overflow-hidden` 到侧边栏
- 第1309行：添加 `flex-shrink-0` 到头部
- 第1322行：添加 `ScrollArea` 开始标签
- 第1565行：添加 `ScrollArea` 结束标签
- 第1568行：修改底部按钮容器样式

**新的结构**：
```tsx
<aside className="w-72 border-r-4 border-black bg-card flex flex-col overflow-hidden">
  {/* 固定头部 */}
  <div className="p-4 border-b-4 border-black/10 flex-shrink-0">
    {/* 标题和返回按钮 */}
  </div>

  {/* 可滚动的中间区域 */}
  <ScrollArea className="flex-1">
    {/* 上传区域 */}
    {/* 色卡选择 */}
    {/* 显示设置 */}
    {/* 杂色过滤设置 */}
    {/* 增强识别设置 */}
  </ScrollArea>

  {/* 固定底部按钮 */}
  <div className="p-4 border-t-4 border-black/10 space-y-2 flex-shrink-0">
    {/* 导出按钮 */}
  </div>
</aside>
```

**测试方法**：
```
1. 打开图纸编辑器
2. 查看左侧操作栏
3. 确认头部（标题）固定不动 ✓
4. 滚动中间区域 → 应该可以滚动 ✓
5. 确认底部（导出按钮）固定不动 ✓
6. 确认可以访问所有配置选项 ✓
```
